Asmc Macro Assembler Reference

### OPTION ELF

**OPTION ELF**:<Value>

Allows to fine-tune the values written to the ELF header if output format ELF.

    OPTION ELF:osabi

The only argument <osabi> will be copied to the ELF header field EI_OSABI.
